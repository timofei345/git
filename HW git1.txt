1. для этого нужно установить git создать локальную директорию на компьютере используя gitbash. Переходим в директорию с помошью cd. Вводим  git clone https://github.com/timofei345/git (git clone ссылка на github).


2. захожу в созданную директорию и ввожу git pull

3. git status --> показывает состояние локальной директории, показывает добавленную и удаленную информацию, локальные изменения	
   git log -->  показывает историю коммитов в обратном порядке начиная с последнего.	
   git remote -v --> отображает список удаленныъ репозиториев с подробной информацией.
